# 🏋️‍♂️ Clube Esportivo - Landing Page

Este projeto é uma landing page responsiva para um clube esportivo, desenvolvida com foco em **UI moderna**, **boas práticas de layout** e **experiência do usuário**.

🎨 Protótipo no Figma

O design da interface foi planejado previamente no Figma, incluindo estrutura de layout, tipografia e composição visual.

Acesse o protótipo:

[https://figma.com/seu-link-aqui](https://www.figma.com/design/ZfVrsKNO097OMlzGzewENd/Sport-Club?node-id=67-92&t=CH1JU6LkVxduQ0Nk-0)

## 🚀 Tecnologias utilizadas

* Next.js
* React
* Tailwind CSS
* TypeScript

---

## ▶️ Como rodar o projeto

### Pré-requisitos

* Node.js (versão 18 ou superior)
* npm ou yarn

### Passo a passo

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/seu-repositorio.git

# Acesse a pasta
cd seu-repositorio

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```

O projeto estará disponível em:

```
http://localhost:3000
```

## 🧠 Decisões técnicas

### 1. Uso do Next.js

Foi escolhido o Next.js por oferecer:

* Otimização automática de imagens (`next/image`)
* Melhor performance
* Estrutura escalável

---

### 2. Estilização com Tailwind CSS

O Tailwind foi utilizado para:

* Acelerar o desenvolvimento
* Manter consistência visual
* Facilitar responsividade

---

### 3. Layout e posicionamento

* Utilização de `flex` e `grid` para estrutura principal
* Uso pontual de `absolute` para sobreposição de elementos (como imagens e overlays)
* Controle de alinhamento com `translate` para centralização precisa

---

### 4. Responsividade

* Breakpoints do Tailwind (`lg`, etc.)
* Ajustes de tipografia e espaçamento para diferentes telas
* Adaptação de layout para manter legibilidade e estética

---

### 5. Componentização

* Separação de componentes como botões (`Button`, `ButtonLink`)
* Código mais reutilizável e organizado

---

### 6. UX e Interações

* Efeitos de hover em botões para melhorar feedback visual
* Uso de sombras (`drop-shadow`) para melhorar contraste
* Gradientes para enriquecer o design

---
